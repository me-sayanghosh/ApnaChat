# ApnaChat

Short full-stack app with a React (Vite) frontend and an Express/MongoDB backend.

## Overview

- Frontend: React + Vite UI with client-side routing
- Backend: Express API with MongoDB and file uploads

## Quick start
### Backend

```bash
cd Backend
npm install
npm run start
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

## Scripts

- Backend: `npm run start`
- Frontend: `npm run dev`
